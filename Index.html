Sobre o projeto

Este projeto tem como objetivo aliar tecnologia e educação em uma mesma direção, auxiliando a todos que desejam acessar o ensino superior de forma gratuita.
Porém, ele vai muito além, visando ajudar, principalmente, aqueles que não apresentam renda suficiente para se preparar para os exames, diminuindo dessa forma a desigualdade.
